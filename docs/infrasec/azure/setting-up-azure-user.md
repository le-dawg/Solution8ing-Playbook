# Setting up your Azure User

This guide will walk you through the process of setting up your Azure user in the Solution8 internal infrastructure.

## Prerequisites

You will need the assistance of someone with administrative privileges in our Azure subscription to help you.

## Steps

1. **Create a user in Microsoft Entra ID:** An administrator will create a new user for you in the Solution8 Microsoft Entra ID.
2. **Assign a license:** The administrator will assign you an Azure license.
3. **Set a temporary password:** The administrator will set a temporary password for you.
4. **Log in to the Azure portal:** Log in to the Azure portal at `https://portal.azure.com` with your new user and temporary password. You will be prompted to change your password.
5. **Set up multi-factor authentication (MFA):** You will be prompted to set up MFA. We recommend using the Microsoft Authenticator app.
6. **Join groups:** The administrator will add you to the appropriate groups in Microsoft Entra ID. These groups will grant you access to the resources you need.
